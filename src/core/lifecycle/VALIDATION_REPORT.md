## 十、业务建议

1. **冷启动判断**：用前24h ROI < 10% 作为"失败"标准，而非"消耗<50元"
2. **关键决策点**：72小时是判断Campaign生死的关键节点
3. **内容为王**：订单收入占比>90%的Campaign才能盈利，广告只是放大器
4. **早期识别**：前6小时ROI>40%的Campaign，92%最终盈利——这是优质内容的信号
